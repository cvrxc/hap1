## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: `pip install matplotlib requests`

## Как запустить
```bash
git clone https://github.com/твой-логин/weather-diary.git
cd weather-diary
python main.py
Как пользоваться
1. При запуске появляется меню:
2. text
1 — Добавить запись
2 — Показать все записи
3 — Фильтровать по дате
3. 4 — Выйти
4. Выбери нужный пункт, введи данные.
5. Данные автоматически сохраняются в data.json
